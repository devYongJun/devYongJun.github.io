### Homebrew 설치 

```bash
# 설치
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# 버전 확인
brew -v
```

### rbenv 설치 

```bash
# 설치
brew install rbenv

# 초기화
rbenv init

# 버전 확인
rbenv -v
```

### ~/.zshrc에 내용 추가

```bash
export PATH={$Home}/.rbenv/bin:$PATH && \
eval "$(rbenv init -)"
```

### Ruby 설치

```bash
# 설치가능한 루비버전 확인
rbenv install -l

# 루비 설치
rbenv install <버전>

# 설치된 루비버전 확인
rbenv versions

# 루비 버전 선택 (global외에 local, shell 옵션도 존재)
rbenv global <버전>
rbenv rehash
```



### Rails 설치

```bash
gem install rails
rbenv rehash
```

### Bundler 설치

```bash
gem install bundler
bundler -v
```
